This plugin in integration with Woocommerce allows you to create different products for rent, with the output of the calendar for booking on the product page.
You can use it for any entity where you need to work with rentals and date ranges.


== Description ==

Whether you're providing service of apartment rental, hotel service, car or kayak, yacht rental or you just a trainer, an event organizer, a couch, etc., try this plugin and easily get reservations on your site with a calendar form and flexible pricing and date intervals options.

= Installation Instructions =

1. Upload `inspirelabs-bookings` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress


== Screenshots ==

1. Choose the product type "Booking"
2. Add new period row
3. Set price per day and choose period rules
4. Repeat no. 2 and 3 if need
5. Check the result on the product page


== Frequently Asked Questions ==

What period rules are available:

1. Any range: customer can book any interval within period
2. Week: customer can book 7-day intervals within period starting from day you set up
3. Whole range: customer can book only whole period you choosed
